1. HTML Headings h1 - h6

Ginagamit para sa hierarchy ng content. h1 = pinaka-important, h6 = least important.

Usage Guide:

Tag Font Size Kailan Gamitin

<h2> 1.75rem / 28px Major sections ng page
<h3> 1.5rem / 24px Sub-sections
<h4> 1.25rem / 20px Sub-sub sections
<h5> 1.1rem / 17.6px Minor headings
<h6> 1rem / 16px Pinaka-maliit, bihira gamitin

2. Paragraph <p>

Ginagamit para sa regular text content. Automatic may margin sa taas at baba.

<p>Ito ay isang paragraph. Pwede mahaba 'to at mag-wrap automatically.</p>

CSS Properties: font-size, line-height, color, text-align, margin

3. Images <img>

Para mag-display ng images. Self-closing tag siya.

<img src="image.jpg" alt="Description" width="300" height="200">

Important Attributes:

  • src - Path ng image - required
  • alt - Description para sa accessibility - required
  • width/height - Size ng image

CSS Properties: width, height, object-fit: cover, border-radius

4. Links/Anchors <a>

Ginagamit para sa hyperlinks papunta sa ibang page or section.

<a href="https://google.com" target="_blank">Visit Google</a>
<a href="#section1">Jump to Section 1</a>

Attributes: href - URL, target="_blank" - open sa new tab

6. Flexbox display: flex

Pinakamadaling way para mag-layout ng items in 1 dimension - row or column.

.container {
  display: flex;              /* I-activate ang flexbox */
  justify-content: center;    /* Center horizontally */
  align-items: center;        /* Center vertically */
  gap: 10px;                  /* Space between items */
}

.item {
  flex: 1;                    /* Equal width lahat */
}

Important Flex Properties:

Property Values Use
justify-content flex-start, center, space-between, space-around Horizontal alignment
align-items flex-start, center, stretch, flex-end Vertical alignment
flex-direction row, column, row-reverse Direction ng items
gap 10px, 1rem Space between items

7. CSS Grid display: grid

For 2D layouts - rows AND columns. Mas powerful sa flexbox for complex layouts.

.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);  /* 3 equal columns */
  gap: 10px;                               /* Space between items */
}

10. Positioning relative | absolute | fixed

Para i-control kung saan lalabas yung elements sa page.

.parent {
  position: relative;  /* Reference point */
}

.child {
  position: absolute;  /* Positioned base sa parent */
  top: 20px;
  left: 50px;
}

.fixed-header {
  position: fixed;     /* Stays sa screen kahit mag-scroll */
  top: 0;
}
Position Behavior
static Default - normal flow
relative Relative sa original position niya
absolute Relative sa nearest positioned parent
fixed Relative sa viewport - di gumagalaw pag scroll
sticky Combination ng relative at fixed

11. Box Model margin | padding | border

Lahat ng HTML element ay box. May 4 parts:

.box {
  margin: 20px;      /* Space sa labas ng border */
  border: 2px solid; /* Border mismo */
  padding: 15px;     /* Space sa loob ng border */
  width: 200px;      /* Content width */
}

Tip: Gamitin mo lagi box-sizing: border-box; para kasama na yung padding at border sa width calculation.

15. HTML5 Page Structure NEW!

Ito yung standard structure ng modern HTML5 pages. Gamitin mo 'to para semantic at SEO-friendly yung code mo.

Basic HTML5 Template:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Title</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <header>
        <nav>
            <!-- Navigation links dito -->
        </nav>
    </header>

    <main>
        <section>
            <h1>Main Content</h1>
            <article>
                <!-- Blog post or main article -->
            </article>
        </section>

        <aside>
            <!-- Sidebar content -->
        </aside>
    </main>

    <footer>
        <p>© 2026 Your Site</p>
    </footer>
</body>
</html>

HTML5 Semantic Tags:

Tag Purpose Kailan Gamitin
<header> Intro content Logo, nav, search bar sa taas ng page
<nav> Navigation links Main menu, breadcrumb, pagination
<main> Main content Unique content ng page - isa lang per page
<section> Thematic grouping Chapters, tabs, or grouped content
<article> Self-contained content Blog posts, news articles, comments
<aside> Sidebar content Related links, ads, author bio
<footer> Footer info Copyright, contact, social links
<figure> Media with caption Images, diagrams with <figcaption>

Why Use Semantic Tags?

  • SEO: Mas maintindihan ng Google yung content mo
  • Accessibility: Screen readers alam kung ano yung nav, main, etc.
  • Maintainability: Mas malinis tignan kaysa puro <div>
  • Standard: Best practice na 'to sa 2026
💡 Tip: Wag ka na gumamit ng <div id="header"> — gamitin mo na <header> mismo!

16. CSS Float INTERACTIVE!

Ginagamit para i-float yung element sa left or right, tapos yung text mag-wrap sa paligid. Old-school layout technique pero useful pa rin sa mga specific cases!

CSS Float Values:

Value Behavior Use Case
left Float sa kaliwa, text sa kanan Image na nasa left side ng paragraph
right Float sa kanan, text sa kaliwa Sidebar, ad box sa right
none Default - walang float I-reset yung float
inherit Manahin sa parent Kapag gusto mo same sa parent element
